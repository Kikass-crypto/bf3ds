What is Brainfuck?

Brainfuck is a simple, esoteric, Turing-complete programming language.

Created in 1993 by Urban Müller, Brainfuck was created so that Müller could try to write the smallest possible compiler. The language was based off of P" and is noted for its lack of practical use.

How does BF3DS work?

BF3DS lets you input code to be run by a Brainfuck interpreter (bf.c (WIP))
